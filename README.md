# Sodoku Solver 

A sodoku solver that works by encoding the sodoku problem as a formula in propositional logic. A SAT solver is then used to solve the problem, and find the solution to the sodoku. The SAT Solver used is my own and can be found [on my github](https://github.com/waqeezaman/CDCL2).


## Solving a large number of Sodoku puzzles

I have also included code to solve a large number of sodoku puzzles in parallel. This greatly speeds up the number of sodoku puzzles that can be solved. This is how I managed to solve the easy problem set, (this contains 100,000 puzzles).
